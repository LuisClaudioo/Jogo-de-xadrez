# Simulação de Movimentos de Xadrez (C)

Projeto desenvolvido em linguagem C para simular os movimentos de peças do jogo de xadrez.

## Tecnologias utilizadas
- Linguagem C
- Execução em terminal

## Descrição do projeto
O programa simula os movimentos das peças Bispo, Torre e Rainha, exibindo no console
as direções possíveis de movimento de cada peça de acordo com regras básicas do xadrez.

## Conceitos aplicados
- Constantes
- Laços de repetição (for)
- Lógica de programação
- Estrutura sequencial
- Saída de dados com printf

## Funcionalidades
- Simulação de movimentos do Bispo
- Simulação de movimentos da Torre
- Simulação de movimentos da Rainha

## Status
Projeto acadêmico para fins de aprendizado.
